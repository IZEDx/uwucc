# uwucc

A bunch of stuff for CC: Tweaked and Create Aeronautics

## Quick Start

### Prerequisites

- Node.js/Bun package manager
- TypeScript-to-Lua compiler setup (already configured in `tsconfig.json`)
- CC:Tweaked computer with Create Aeronautics, CC: Sable

### Building

```bash
# Install dependencies
bun install
# or
npm install

# Build the project
bun run build
# or
npm run build

# Watch mode (auto-rebuild on changes)
bun run dev
# or
npm run dev
```
